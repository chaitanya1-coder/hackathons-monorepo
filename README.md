### Option 1: Starting a New Project

1. **Clone this repository**
   ```bash
   git clone <repository-url>
   cd hackathons
   ```

2. **Create your submission directory**
   ```bash
   mkdir submissions/your-team-name-or-project-name
   cd submissions/your-team-name-or-project-name
   ```

3. **Start building!**
   - Add your project files to this directory
   - Include a `README.md` in your submission folder with project details
   - Follow the submission guidelines below

4. **Submit your project**
   ```bash
   git add .
   git commit -m "Add submission: [Your Project Name]"
   git push origin Stellar-x-Polkadot-HackerHouse-BLR
   ```

### Option 2: Pushing Existing Code

1. **Clone this repository**
   ```bash
   git clone <repository-url>
   cd hackathons
   ```

2. **Copy your existing project**
   ```bash
   cp -r /path/to/your/project submissions/your-team-name-or-project-name
   ```

3. **Add submission metadata**
   - Create a `README.md` in your submission folder (see template in `submissions/TEMPLATE/`)
   - Ensure your project follows the submission guidelines

4. **Submit your project**
   ```bash
   git add .
   git commit -m "Add submission: [Your Project Name]"
   git push origin main
   ```
